LAN Simulation Console

ThreadSync LAN Chat for Multithreading Demonstrations

A real-time chat web app that visualizes thread assignment, active users, synchronization stats, and event logs while users communicate in room-isolated channels.

How to Use

Key Features

Real-Time Messaging

Low-latency message broadcasting over Socket.IO with sender tags and timestamps.

Thread Visualization

Live thread pool state indicators show idle and busy execution paths.

Presence Tracking

Connected nodes list updates instantly with user identity and assigned thread ID.

Kernel Logs

Operational logs display system, chat, and warning events in a terminal-style view.

Command Rename

Users can rename themselves in-chat using the /name command format.

Responsive Layout

Dashboard adapts to desktop, tablet, and mobile with stacked interaction panels.

How to Use

  1. 1. Start the serverRun node server.js (or npm start) and open http://localhost:3000 in one or more browsers on your LAN.
  2. 2. Enter credentialsProvide any SSID and password in the V-LAN access card to establish a room-isolated session.
  3. 3. Send and receive chatType in the message box and press Enter or send. New messages auto-scroll in the feed.
  4. 4. Rename your nodeUse /name your_name in the chat input to update your display name instantly.
  5. 5. Observe synchronizationTrack active users, request count, thread states, and network hash in real time.
  6. 6. Review system logsUse the kernel logs panel to inspect chat and system events during your demo session.
